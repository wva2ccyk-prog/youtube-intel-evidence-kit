# Synthetic Topic Demo

This fixture is a public-safe cross-video demo. It is not real YouTube data.

The three synthetic videos describe the same orchard sensor topic from different positions:

- a vendor-style briefing
- a field agronomist walk-through
- a local policy panel

The expected output is not a better single-video summary. The expected output is a TopicCollection that maps repeated claims, disagreements, outliers, source videos, speakers, timestamps, and evidence coordinates.
